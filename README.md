# Rad Electric Co - Marketing Website

Professional marketing website for Rad Electric Co, showcasing electrical and low voltage services.

## Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Development Server
```bash
npm run dev
```

Your site will open at **http://localhost:5173**

### 3. Build for Production
```bash
npm run build
```

## Technology Stack

- **React 18** - Component-based UI framework
- **TypeScript** - Type-safe JavaScript
- **Vite** - Fast build tool and development server
- **React Router** - Client-side routing
- **Tailwind CSS v4** - Utility-first CSS framework
- **Motion (Framer Motion)** - Animation library
- **Lucide React** - Icon library
- **Netlify Forms** - Form submission backend

## Deployment

### Netlify
1. Connect repository to Netlify
2. Build command: `npm run build`
3. Publish directory: `dist`
4. Forms will automatically work with Netlify's backend

### Vercel
1. Import repository
2. Framework preset: Vite
3. Build command: `npm run build`
4. Output directory: `dist`
